# Паттерны Autonomous Loops

> Источник: https://github.com/affaan-m/everything-claude-code

## 6 паттернов

### 1. Sequential Pipeline (`claude -p`)
Цепочка неинтерактивных вызовов. Каждый шаг — отдельный `claude -p` с фокусной задачей.
```bash
claude -p "Проанализируй канбан" | claude -p "Предложи приоритеты"
```
**Для нас:** простые цепочки типа "прочитай → проанализируй → запиши".

### 2. NanoClaw REPL
Persistent session-aware loop. Сохраняет историю разговора между перезапусками.
**Для нас:** основа для Продуктолога в фоновом режиме.

### 3. Infinite Agentic Loop
Два промпта: оркестратор + параллельные подагенты. Оркестратор разбивает задачу, агенты выполняют.
**Для нас:** когда Продуктолог захочет запустить market-researcher + product-strategist параллельно.

### 4. Continuous Claude PR Loop
Создаёт PR → мониторит CI → автофикс при ошибках → мёрж. Итеративно.
**Для нас:** автоматизация деплоя Genealogy.

### 5. De-Sloppify Pattern
Отдельный проход очистки после реализации. Убирает избыточность в тестах и коде.
**Для нас:** после каждого MVP-спринта — чистка.

### 6. Ralphinho / RFC-Driven DAG
Спецификация → граф зависимостей → параллельное выполнение в worktrees → merge queue.
**Для нас:** тяжёлая артиллерия для крупных фич.

## Матрица выбора

| Задача | Паттерн |
|--------|---------|
| Простая цепочка | Sequential Pipeline |
| Фоновый мониторинг | NanoClaw REPL |
| Параллельный анализ | Infinite Agentic Loop |
| CI/CD автоматизация | Continuous PR Loop |
| Пост-спринт очистка | De-Sloppify |
| Крупная фича с зависимостями | Ralphinho DAG |
