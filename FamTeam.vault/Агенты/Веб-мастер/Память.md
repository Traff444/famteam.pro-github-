## Уроки (что пошло не так)

| Проблема | Урок |
|----------|------|
| Netlify Function 500 | Netlify Functions v2 (ESM) используют `process.env`, не `Netlify.env.get()` |
| Puppeteer на CI | Netlify CI не имеет Chrome → prerender только локально |
| GitHub Pages 404 | SPA на GitHub Pages требует 404.html + redirect hack |
| Vite noscript в head | Vite парсер ломается на `<noscript><iframe>` в `<head>` → перенести в `<body>` |
| GitHub Pages base path | Без custom domain ассеты ищут от корня → нужен `base` в vite.config.ts |
