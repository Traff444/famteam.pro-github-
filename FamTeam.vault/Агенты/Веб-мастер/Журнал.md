**🔴 КРИТИЧЕСКИЕ ПРОБЛЕМЫ (5 шт, блокируют деплой):**

   - manifest.json — для PWA

   - hero-tree.png: 1.3 MB
   - hero-paper-texture.png: 2.0 MB
   - Итого: 3.3 MB на странице

   → LCP > 4 сек (нужно < 2.5 сек) = штраф Google

   - Только Organization, нет Product schema для цены
   - Нет FAQPage, нет BreadcrumbList

   - Нет информации об авторе/команде
   - Нет testimonials
   - Нет примеров результатов

5. **Контент для SEO:**
   - H1 не на странице
   - Нет internal linking между страницами
   - Структурированная информация минимальна

**🟠 ВЫСОКИЙ ПРИОРИТЕТ (2 шт):**

- Lazy loading для больших изображений
