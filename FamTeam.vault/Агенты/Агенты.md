# Агенты

17 агентов Claude Code, организованных по фазам разработки. Вызов: `Agent tool` с `subagent_type`.

## Фаза 1 — Идея и исследования

| Агент | Тип | Описание | Когда вызывать |
|-------|-----|----------|----------------|
| [[product-strategist]] | PM | Стратегия продукта, бизнес-модели, ценообразование, SWOT, Porter | Новый продукт, стратегия, ценообразование |
| [[market-researcher]] | PM | Анализ рынка, конкуренты, персоны, TAM/SAM/SOM | Анализ конкурентов, размер рынка |
| [[product-discovery]] | PM | Исследование пользователей, приоритизация фич, эксперименты | Интервью, приоритизация бэклога |

## Фаза 2 — Планирование и архитектура

| Агент | Тип | Описание | Когда вызывать |
|-------|-----|----------|----------------|
| [[pm-executor]] | PM | PRD, user stories, спринты, OKR, roadmap, SQL | PRD, планирование спринта |
| [[planner]] | DEV | Планирование сложных фич и рефакторинга | Сложная фича, перед кодом |
| [[architect]] | DEV | Архитектура системы, масштабирование, тех. решения | Архитектурные решения |

## Фаза 3 — Разработка

| Агент | Тип | Описание | Когда вызывать |
|-------|-----|----------|----------------|
| [[tdd-guide]] | DEV | TDD: сначала тесты, потом код. 80%+ покрытие | Любая фича или багфикс |
| [[code-reviewer]] | DEV | Код-ревью: качество, безопасность, поддерживаемость | После написания кода |
| [[build-error-resolver]] | DEV | Исправление ошибок сборки и типов | Билд сломался |
| [[database-reviewer]] | DEV | PostgreSQL: оптимизация запросов, схема, индексы | Работа с БД |
| [[security-reviewer]] | DEV | Уязвимости: секреты, инъекции, OWASP Top 10 | Чувствительный код |
| [[go-reviewer]] | DEV | Go код-ревью: идиоматичность, конкурентность | Go-код |
| [[go-build-resolver]] | DEV | Исправление ошибок сборки Go, go vet, линтер | Go билд сломался |
