## 1. Gramps

### Что это
Самый мощный бесплатный генеалогический инструмент в мире. Desktop-приложение (Python/GTK) + Web-companion (Lit/Flask). Open source, GPL-2.0. Существует с 2001 года.

### Сильные стороны
- Глубочайшая модель данных: персоны, семьи, источники, события, места, медиа, репозитории
- 20+ типов отчётов, несколько форматов вывода (PDF, HTML, ODT, SVG)
- Первоклассная GEDCOM-поддержка
- Расширяемость через плагины
- Gramps Web: AI Chat Assistant (pydantic-ai + OpenAI-compatible API)
- Privacy-first: данные у пользователя
- 65 языков, активное сообщество

### Слабые стороны
- UX уровня 2005 года, высокий барьер входа
- Self-hosted, нет SaaS
- PDF визуально бедные, нет premium-оформления
- AI только реактивный (отвечает на вопросы по дереву), не ищет информацию
- Нет белорусского языка
- Нет специализации на восточноевропейских архивах

### Угроза для нас: НИЗКАЯ
Gramps обслуживает совершенно другой сегмент: технически грамотные серьёзные генеалоги, готовые тратить время на установку и ручной ввод. Наш клиент хочет результат за $5 за минуты.

### Чему учиться
- Глубина модели данных как эталон для будущих версий
- GEDCOM-совместимость важна для миграции пользователей
- Их AI Chat в Web-версии показывает направление развития (но они реактивные, мы проактивные)

## 2. webtrees

### Что это
Веб-приложение для совместного ведения генеалогии. PHP/MySQL, self-hosted. Open source, GPL-3.0. С 2013 года, 8 453 активных установки.

### Сильные стороны
- 10+ типов визуализации деревьев (fan chart, hourglass, pedigree map)
- 15+ типов отчётов с PDF-экспортом
- Совместная работа нескольких пользователей
- Гибкие права доступа
- 75 языков включая русский
- Стабильная кодовая база, 13 лет разработки

### Слабые стороны
- Требует PHP-хостинг, нет SaaS
- UI устарел
- Нет AI вообще
- PDF утилитарные, чёрно-белые
- Нет белорусского языка
- Нет специализации по архивам

### Угроза для нас: НИЗКАЯ
Ещё более узкий сегмент чем Gramps: те, у кого есть свой PHP-хостинг. Нулевое пересечение с нашей аудиторией.

### Чему учиться
- Богатые визуализации (fan chart, hourglass) как фичи для будущих версий
- Система прав доступа для семейного доступа к деревьям

## 3. MyMap.AI Family Tree Maker

### Что это
AI-powered SaaS для визуального мышления. Family Tree Maker это один из бесплатных SEO-tools, а не основной продукт. Команда 2 человека, ~$152K ARR, ~500K пользователей.

### Сильные стороны
- Мгновенная генерация дерева из текста (чат с AI)
- No-signup, бесплатный вход
- Real-time collaboration
- Мультимодельный AI (GPT-4, Claude, Gemini, Qwen)
- Хорошая SEO-стратегия через бесплатные tools

### Слабые стороны
- Не генеалогический инструмент по сути, а diagramming tool
- Нет GEDCOM
- Нет структурированных данных (даты, места, источники)
- Нет архивных баз, нет исследовательской функции
- PDF это screenshot, не structured report
- Нет русского/белорусского языка
- Жалобы на биллинг

### Угроза для нас: СРЕДНЯЯ
MyMap перехватывает SEO-трафик по запросам "family tree maker". Если пользователь ищет "сделать генеалогическое дерево онлайн", он может попасть к ним, а не к нам. Но их продукт не решает задачу исследования, ценовая разница значительна ($20/мес vs $5 одноразово).

### Чему учиться
- No-signup бесплатный инструмент как SEO-magnet (мы можем сделать бесплатный генератор простого дерева)
- Chat-first интерфейс для ввода данных
- Мультимодельный AI backend

| BY/RU архивная специализация | -- | -- | -- | ++ |
| Цена < $10 | ++ (бесплатно) | ++ (бесплатно) | - ($20/мес) | ++ ($5) |
| Без установки, без хостинга | -- | -- | ++ | ++ |
| Белорусский/русский язык | +/- | +/- | -- | ++ |
| AI-исследование | - (только chat) | -- | +/- (визуализация) | ++ |
| Письма в архивы | -- | -- | -- | ++ |

### 2. Open source инструменты (Gramps, webtrees) не конкуренты, а **complementary**

Пользователь может:
1. Купить наш PDF за $5 (entry point)
2. Заинтересоваться генеалогией глубже
3. Перейти на Gramps/webtrees для серьёзного ведения дерева
4. Вернуться к нам за $19/$49 для углублённого AI-исследования

Мы не конкурируем, а создаём воронку в экосистему.

### 3. MyMap это SEO-угроза, но не продуктовая

Они перехватывают top-of-funnel трафик. Ответ: создать свой бесплатный инструмент визуализации дерева (без регистрации) как SEO-magnet.

### 4. Уникальные дифференциаторы FamTeam

1. **AI-исследование** (не визуализация, а поиск предков)
2. **Premium PDF** (подарочный артефакт, не утилитарный отчёт)
3. **Готовые письма в архивы** (уникальная фича, нет ни у кого)
4. **BY/RU архивная специализация** (знание фондов, губерний, форматов запросов)
5. **$5 одноразово** (не подписка, не €200)
6. **10% на помощь животным** (emotional trigger)
