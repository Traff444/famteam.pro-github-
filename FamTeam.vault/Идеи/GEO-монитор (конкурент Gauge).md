# GEO-монитор — AI Visibility Platform

> Конкурент Gauge (withgauge.com, $100-599/мес, SF). Мониторинг видимости бренда в AI-поисковиках.

## Суть

Ежедневно прогоняем промпты через ChatGPT/Claude/Gemini/Perplexity → проверяем упоминается ли бренд клиента → gap-анализ с конкурентами → генерируем контент для улучшения видимости.

## Метрики продукта

- **Mention Rate** — % ответов AI где упомянут бренд
- **Citation Rate** — % ответов где цитируется сайт
- **Referral Traffic** — трафик с AI-платформ (GA4)
- **Gap Score** — промпты где конкурент есть, клиента нет

## Что нужно построить

1. **Crawler** — скрипт прогона промптов через API моделей (ChatGPT, Claude, Perplexity)
2. **Parser** — извлечение упоминаний бренда и источников из ответов
3. **DB-схема** — BrandTrack, PromptResult, CitationSource
4. **Dashboard** — страница в admin panel или отдельный SaaS
5. **Рекомендательный движок** — gap → задачи для Контент-мейкера
6. **GA4/GSC API** — подключение аналитики клиента

## Стратегия входа

1. **Фаза 0** (сейчас) — Веб-мастер мониторит видимость FamTeam/Genealogy руками → набираем экспертизу
2. **Фаза 1** (после Genealogy launch) — Автоматизация для себя: cron + парсер + дашборд
3. **Фаза 2** — Открываем как продукт/фичу awork для клиентов
4. **Фаза 3** — SaaS с self-serve подпиской
