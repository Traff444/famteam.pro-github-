# VetChat — ветеринарная телемедицина BY/RU

> Референс: PetCentric Health ($49/мес, $600K raised). Рынок $747M → $3.3B к 2034, CAGR 18%.

## Суть

Telegram-бот + веб-платформа: владелец питомца описывает проблему → AI-триаж оценивает срочность → подключает дежурного ветеринара (видео/текст) или направляет в клинику. Подписка + разовые консультации + аптека/корм.

## Почему именно мы

| Актив FamTeam | Как используется |
|---------------|-----------------|
| **Talkify** (production-ready) | Платформа: омниканальность, CRM, лид-скоринг, воронки |
| Telegram-боты | @teamfam_bot, ZeroStaff, Aiden — опыт |
| Claude API | AI-триаж: оценка срочности, первичные рекомендации |
| PayPal биллинг | Подписки (делаем для Genealogy) |
| VPS (212.227.58.60) | Инфраструктура |
| Genealogy "10% животным" | Мост к pet-аудитории, общий бренд |

## Модель

```
Владелец → Telegram/Web
  → AI-триаж (Talkify + Claude):
    - Срочность 0-10
    - Срочно (8+) → "Едьте в клинику [адрес], пока едете: [инструкция]"
    - Не срочно → Дежурный ветеринар (видео/текст)
  → Ветеринар консультирует
  → Рецепт → Аптека/корм (доставка через партнёров)
```

## Рынок

- Глобальный рынок вет-телемедицины: $747M (2025) → $3.3B (2034), CAGR 18%
- 80% вопросов решаются дистанционно (Airvet)
- В Москве/Питере дефицит ветеринаров экзотов: запись на 2-3 недели
- Ниша "телемедицина для экзотов" — пуста
- Зумеры: питомцы = дети, готовы платить за удобство

## Ниши входа

1. **BY: экзоты** — попугаи, рептилии, хорьки. Специалистов почти нет, запись на недели
2. **RU: корпоративный B2B** — пет-бенефит для офисов (как Airvet в США)
3. **Общий: ночные/выходные** — когда клиники закрыты, $49/мес за спокойствие
